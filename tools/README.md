# tools/ — провенанс и проверка базы переводов

## Что это

`fetch_official_names.py` — конвейер, который тянет локализованные имена напрямую из
официальных клиентских данных Blizzard через CSV-экспорт [wago.tools](https://wago.tools)
(enUS + ruRU для конкретного билда) и сверяет их с базой аддона.

Назначение — **доказуемое происхождение данных**: показать, что имена в Matreshka — это
официальный перевод Blizzard под нужный билд, а не произвольный сторонний источник.

```
python tools/fetch_official_names.py ItemSparse 2.5.5.68101 Display_lang \
    Matreshka/Database/Items/ru.lua Matreshka/Database/Items/en.lua
```
